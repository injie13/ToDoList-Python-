To-Do List Application

Description
The To-Do List Application is a simple, text-based Python program that allows users to manage tasks effectively. Users can add tasks, delete tasks by their number, view the list of tasks, and exit the program when finished.

Features
Add a Task: Add a new task to the to-do list.
Delete a Task: Remove a specific task by entering its corresponding number.
List Tasks: View all tasks currently in the to-do list.
Quit: Exit the application.


How to Use

Run the Python script in your terminal or IDE.
Follow the menu options to manage your tasks:
Enter 1 to add a new task.
Enter 2 to delete an existing task.
Enter 3 to view all tasks.
Enter 4 to quit the application.


Provide inputs as prompted by the program.

Code Structure
addTask(): Prompts the user for a task description and adds it to the list.
deleteTask(): Displays the list of tasks and allows the user to remove a task by its number.
listTasks(): Prints all tasks with their respective indices for easy reference.

Main Program:
Displays a menu for user options.
Continuously loops until the user chooses to quit.
Requirements
Python 3.19 installed on your system.
